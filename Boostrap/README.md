# PracticaCSSFinal

## Descripción del proyecto

PracticaCSSFinal es un sitio web educativo desarrollado con HTML5 y CSS3 cuyo objetivo es poner en práctica los conceptos fundamentales del desarrollo web. El proyecto incluye el uso de etiquetas semánticas, formularios, tablas, contenido multimedia, navegación entre páginas y estilos personalizados mediante hojas de estilo externas.

Además, se aplicaron principios de diseño responsivo para que el sitio pueda adaptarse correctamente a diferentes tamaños de pantalla, mejorando la experiencia del usuario tanto en computadoras como en dispositivos móviles.

La ampliación del proyecto incorpora una nueva página llamada **portafolio.html**, donde se presentan proyectos realizados, habilidades desarrolladas y resultados de aprendizaje relacionados con el estudio de HTML y CSS.
