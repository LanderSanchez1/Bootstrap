# Portafolio Web - Lander Sanchez

## Descripción General

Este proyecto es un portafolio web personal desarrollado como parte de las prácticas de la materia de Fundamentos Web / Diseño Web en la **Universidad de las Fuerzas Armadas ESPE**, carrera de **Tecnologías de la Información**.

El sitio está compuesto por varias páginas que muestran mi perfil, mi portafolio de proyectos, mi proyección personal, académica y profesional, una sección de búsqueda con formularios, y una página de contacto. Las primeras páginas fueron construidas con HTML semántico y CSS puro, mientras que la página de **Proyección** fue desarrollada utilizando **Bootstrap 5**, aplicando un enfoque *Mobile First* y personalización mediante CSS propio.

## Tecnologías Utilizadas

- **HTML5** – estructura semántica del contenido (header, nav, main, section, article, aside, footer, figure).
- **CSS** – estilos generales y específicos por página, animaciones, diseño responsivo (Mobile First).
- **Bootstrap 5** – sistema de grid, componentes (navbar, cards, accordion, carousel, modal, badges, progress bars, list groups, tablas, formularios).
- **Font Awesome 6** – iconografía.

## Instrucciones para Visualizar el Sitio

1. Clonar o descargar este repositorio:
   ```bash
   git clone https://github.com/usuario/nombre-del-repositorio.git
   ```
2. Abrir la carpeta del proyecto en un editor de código (por ejemplo, Visual Studio Code).
3. Abrir el archivo `index.html` con un navegador web (recomendado: usar la extensión **Live Server** de VS Code para evitar problemas con rutas relativas).
4. Desde la barra de navegación se puede acceder a las demás secciones: Portafolio, Proyección, Buscar y Contacto.
5. La página `proyeccion.html` requiere conexión a internet, ya que utiliza Bootstrap 5, Font Awesome y algunas imágenes alojadas externamente (Unsplash) a través de CDN.
